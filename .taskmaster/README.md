
## 🚀 시작하기

Task Master가 초기화되었습니다! 다음 명령어를 사용하세요:

- `/tm status` - 현재 상태 확인
- `/tm add [작업]` - 새 작업 추가
- `/tm list` - 작업 목록 보기
- `/tm analyze` - 프로젝트 분석
